# Business Request - Cisco Duo

## Request

The security team requested Cisco Duo integration with Microsoft Entra ID to improve MFA enforcement and prepare the environment for Zero Trust access policies.

## Scope

- Cisco Duo Single Sign-On
- Microsoft Entra authorization
- Admin consent
- Enterprise Application validation
- Audit log review

## Out of Scope

- Full Conditional Access policy deployment
- Production rollout
- Device Trust enforcement
- End-user MFA testing

These items will be handled in IAM-004 Conditional Access and Zero Trust.
